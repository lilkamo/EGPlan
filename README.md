# GlowUp Glutes — 8-Week Home Program

A mobile-first Progressive Web App (PWA) for an 8-week glute and abs home training program.

## Features
- 5 sections: Home, Workouts, Nutrition, Gear, Progress
- Full workout breakdowns for all 4 phases (Weeks 1–2, 3–4, 5–6, 7–8)
- Every exercise explained with coaching cues and tips
- Nutrition targets in US measurements (oz, lbs, calories)
- Equipment guide with US pricing
- Progression tracking and red flags
- Installable to home screen as an app (iOS and Android)

## Deploy to GitHub Pages

1. Upload all files to a GitHub repository
2. Go to Settings → Pages → Source → Deploy from main branch / root
3. Wait ~2 minutes, then visit `https://yourusername.github.io/your-repo-name`

## Install to Home Screen

**iPhone/iPad (Safari only):**
1. Open the URL in Safari
2. Tap the Share button (square with arrow)
3. Tap "Add to Home Screen"
4. Tap "Add"

**Android (Chrome):**
1. Open the URL in Chrome
2. Tap the three-dot menu
3. Tap "Add to Home Screen" or "Install App"

## Files
- `index.html` — main app (everything is in one file)
- `manifest.json` — PWA manifest for installability
- `sw.js` — service worker for offline support
- `icon-192.svg` / `icon-512.svg` — app icons (rename to .png if needed)
